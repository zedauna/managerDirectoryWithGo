# managerDirectoryWithGo

Tools in Go for managing Directory

## Fonctionnalités
- Recherche récursive
- Support multiplateforme
- Confirmation interactive
- Journalisation des opérations

## Installation
```bash
go clone https://github.com/zedauna/managerDirectoryWithGo.git
```

## Utilisation
```bash
================================================================
Arborescence
================================================================
Usage: go run .\cmd\arborescence [répertoire] [stockage] <unit> <prefix> <extensions>

go run .\cmd\arborescence P:\870 D:\aborescence partage_psim go

go run .\cmd\arborescence H:\doss\x870000\12_portail_sig D:\aborescence MB silo_portail_sig "csv,py,xlsx,sql"


================================================================
Extension
================================================================
go run .\cmd\changeExtension\

================================================================
Remove (with wilcard)
================================================================
go run .\cmd\remove\

```
